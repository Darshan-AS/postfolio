## Reasoning

- Code generation tools (`json_serializable` and `slang`) drastically cut down on boilerplate while preserving complete compile-time type safety.
- The UX for adding Nominees is greatly enhanced by offering predefined, localized relationship options in a dropdown instead of requiring manual text entry every time.
