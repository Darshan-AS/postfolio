# Session Log - 2026-05-31

## Context
Fixed a layout issue where white space appeared above the keyboard in Demo Mode and standardized nested scaffold behavior.

## Changes Made
- **Main Shell**: Replaced the manual `MediaQuery` hack with the official Flutter best practice. Wrapped the inner navigation shell in a `Builder` to obtain a context from *inside* the Scaffold's body.
- **Conventions**: Updated `conventions.md` to mandate using a `Builder` when modifying `MediaQuery` for inner pages in a shell architecture, preventing outer contexts from forcing raw keyboard insets into inner pages.

## Decisions
- Discovered the root cause of the "Double Inset" bug: `MediaQuery.removePadding` was using the outer `MainShellScaffold` context, bypassing the `Scaffold`'s native inset consumption and forcing the raw 300px keyboard inset back into the inner page. Utilizing a `Builder` fixes this natively without any manual offset calculations.
